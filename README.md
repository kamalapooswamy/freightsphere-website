# Static Site (starter)

Simple static website scaffold (HTML, CSS, JS) created by the assistant.

Quick start

1. Open a terminal in the `static-site` folder.
2. Serve the site locally. If you have `npx` available you can run:

```bash
npx serve . -l 5000
```

Or use a global tool like `live-server`:

```bash
live-server
```

Notes

- The site is intentionally minimal. Add pages in the root and link from `index.html`.
- To add a build step, introduce a bundler or static site generator and update `package.json`.
